Case Study

The Goal

The goal for this praticular webpage was to update it to have a modern look and feel. As well as make the webpage responsive on mobile devices.

The Tools

  • Teamsite
  • Robohead
  • Adobe Dreamweaver
  • HTML5/CSS3/JavaScript
  • Bootstrap
  • Adobe Photoshop

The Process

I received the initial request via Robohead to update the older version of the Seasoned Loan Offerings webpage. I was provided a rough wireframe of what client would like the page to look, as well as an accompanying Excel Spreadsheet that contained the graphics for the updated webpage.

First, I had to clip the provided graphic images from the excel sheet and size them properly for the web page using Adobe Photoshop. Once the graphics were ready for the web, I designed a prototype of the webpage based on their provided criteria (Grid based layout, graph launching into a modal onclick, and mobile first design) and sent them a link over for review.

Lastly, once the prototype was approved in review, I sent the staged web page over to QA for any last minute additions or changes.

The Challenges

One of the biggest issues was building it without having an approved design. Usually, the development team is provided an approved design from the web design team before building a webpage, especially for the main site. However, for this webpage, the client wanted it up and running by Freddie Mac's 50 year anniversary.

In order to overcome this challenge, I was in close communication with the client. Providing prototypes and quickly iterating on the page based on their feedback on the previous prototype.

The Result

The page launched without a hitch in preparation for Freddie Mac's anniversary. The page presented to the user was modern and easy to use on desktop and mobile. It satisfies the client's goals for the look and feel of the page, as well as it's functionality.
