Case Study

Seasoned Loan Offerings

The Goal

The goal for this praticular webpage was to update it to have a modern look and feel. As well as make the webpage responsive on mobile devices.

The Tools

  • Teamsite
  • Robohead
  • Adobe Dreamweaver
  • HTML5/CSS3/JavaScript
  • Bootstrap
  • Adobe Photoshop

The Process

I received the initial request via Robohead to update the older version of the Seasoned Loan Offerings webpage. I was provided a rough wireframe of what client would like the page to look, as well as an accompanying Excel Spreadsheet that contained the graphics for the updated webpage.

First, I had to clip the provided graphic images from the excel sheet and size them properly for the web page using Adobe Photoshop. Once the graphics were ready for the web, I designed a prototype of the webpage based on their provided criteria (Grid based layout, graph launching into a modal onclick, and mobile first design) and sent them a link over for review.

Lastly, once the prototype was approved in review, I sent the staged web page over to QA for any last minute additions or changes.

The Challenges

One of the biggest issues was building it without having an approved design. Usually, the development team is provided an approved design from the web design team before building a webpage, especially for the main site. However, for this webpage, the client wanted it up and running by Freddie Mac's 50 year anniversary.

In order to overcome this challenge, I was in close communication with the client. Providing prototypes and quickly iterating on the page based on their feedback on the previous prototype.

The Result

The page launched without a hitch in preparation for Freddie Mac's anniversary. The page presented to the user was modern and easy to use on desktop and mobile. It satisfies the client's goals for the look and feel of the page, as well as it's functionality.

Strength in Stability Campaign

The Goal

The goal for this page was to redesign the current companies' about section to reflect the changes that Freddie Mac had gone through over the past 50 years and where it was going in the future.

The Tools

  • Teamsite
  • Robohead
  • Adobe Dreamweaver
  • HTML5/CSS3/JavaScript
  • Bootstrap

The Process

This redesign was one of the biggest projects the web development team worked on that year. In the drafting stagings of the design of the web page, the web design team, communication team, back end development, and front end development would hold bi weekly stand ups to go over prototypes from the design team and brainstorm ideas with the communications.

After 4 weeks of prototyping and brainstorm, we were able to come up with a web design that was functional from a JavaScript perspective and appealing to the communication teams vision.

My role in this project was to develop the content area of the webpage. Including the button interface, typography of the page, and the grid based layout.

The Challenges

One of the challenges that arrived during development was getting the web designs of the buttons on the page to look the same on the web.

This was my first time using the 'position' CSS property to place and overlap the inverted carrot icon on top of the button image. I was able to overcome this challenge, by puzzling out the problem and using online resources to solve my problem. However, I also reached out to one of the senior developers to review and QA my solution, before pushing it over to production.

The Result

The redesign of the Strength in Stability page, originally about the company, was received incredibly well by other employees and executives at Freddie Mac. The page is still iterated on and updated at the end of each quarterly financial period.
